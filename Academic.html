Undergraduate Programs



Software Engineering


IT and Systems

What will I study?

Throughout the Software Engineering program, there will be a focus on Software design, requirement study, programming and database-related courses. Major course areas are:

  • Software Design and Architecture
  • Software Requirements Engineering
  • Mobile Application Development
  • Web Application Development
  • Software Project Management
  • Database Models
  • Programming / Coding
  • Data Structures and Algorithms

Graduate Requirements

Grade

To Graduate from this program, students must pass every compulsory course and support course, as well as 9 credit hours of elective courses. The cumulative grade point average should be at least 2.0.

Internships

An internship is compulsory non-credit work for a student to Graduate from this program. It is also an opportunity for students to achieve personal learning goals.

Community Service

A student is recommended to complete 24 hours of community service before graduation. This is an opportunity for students to give back to the community.

Career Paths

Many career opportunities are available to graduates of the Software Engineering program. They will have the potential to work in various roles such as:

  • Software Designers
  • Software Developer
  • Requirements Analyst
  • Software Project Manager
  • Usability Experts
